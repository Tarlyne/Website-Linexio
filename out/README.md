# Public Ordner

In diesen Ordner gehören alle statischen Dateien, die direkt öffentlich zugänglich sein sollen.

## Deine Aufgaben:
1. Kopiere dein Logo hierher und nenne es **`logo.png`**.
   (Dann wird es automatisch in der Navbar angezeigt).

2. Kopiere deine Screenshots in den Unterordner `screenshots/`.
